# Update - v3.2.1

## 追加要素

### 個人チェストが登場

* 自分専用のチェストが登場しました。
  * `/personal-chests` (alias: `/pc`) で自分専用の個人チェスト(エンダーチェスト)が使えます。
  * ここに入れたアイテムは自分以外アクセスすることは出来ません。
    * ただし、運営はアクセスすることができます。

![pc.png](images/pc.png)

* 参加時のサブタイトルにキャッチコピーが表示されるようになりました。
